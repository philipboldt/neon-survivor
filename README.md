# Neon Survivor

A fast-paced, survival-style arcade game with a vibrant neon aesthetic. Survive as long as you can against increasing waves of enemies, collect experience, find hidden treasures, and upgrade your arsenal to become unstoppable.

![Neon Survivor Screenshot Placeholder](https://via.placeholder.com/1000x1000.png?text=Neon+Survivor+Gameplay)

## 🌟 Features

- **Neon Aesthetics:** High-contrast visuals with intense glowing effects and a deep-space atmosphere.
- **Dynamic Difficulty:** Enemy spawn rates accelerate every 15 seconds, pushing your reflexes to the limit.
- **Epic Mini-Bosses:** Face off against giant neon adversaries every 45 seconds—be prepared for the fight!
- **Destructible Environment:** Hunt for **Neon Brown Boxes** in the corners of the world to find game-changing items.
- **Special Pickups:**
  - 🧲 **Magnet Dot:** Instantly pull all experience dots on the map towards you.
  - 💰 **Gold Dot:** Accumulate wealth to track your survival success.
  - 💚 **Heal Dot:** Emergency restoration to full health (limited 30s lifespan!).
- **Upgradable Arsenal:**
  - **Attack Count:** Fire more projectiles simultaneously.
  - **Projectile Damage:** Tear through enemies faster.
  - **Circling Balls:** Deploy blue neon orbitals that rotate around you, damaging anything they touch.
- **Optimized Engine:** 
  - **Tiled Starfield:** Pre-rendered background tiles for silky smooth motion.
  - **Spatial Partitioning:** Efficient grid-based collision and targeting for hundreds of entities.
  - **Region-based Rendering:** High performance even when the screen is full of chaos.

## 🕹️ Controls

- **Movement:** `WASD` or Arrow Keys
- **Pause:** `SPACE`
- **Restart:** Press `SPACE` or Click the Screen after Game Over

## 🛠️ Tech Stack

- **Frontend:** HTML5 Canvas, Vanilla JavaScript (ES6+), Vanilla CSS
- **Performance:** Custom Spatial Grid & Sprite Pre-rendering
- **Hosting Ready:** Fully compatible with GitHub Pages

## 🚀 Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/philipboldt/neon-survivor.git
   cd neon-survivor
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run serve
   ```

4. **Play the game:**
   Open your browser and navigate to `http://localhost:3000`.
