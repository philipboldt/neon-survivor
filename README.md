# Neon Survivor

A fast-paced, survival-style arcade game with a vibrant neon aesthetic. Survive as long as you can against increasing waves of enemies, collect experience, and upgrade your weapon to become unstoppable.

![Neon Survivor Screenshot Placeholder](https://via.placeholder.com/1000x1000.png?text=Neon+Survivor+Gameplay)

## 🌟 Features

- **Neon Aesthetics:** High-contrast visuals with glowing effects and a space-themed starfield.
- **Dynamic Difficulty:** Enemy spawn rates increase every 15 seconds, challenging your survival skills.
- **Weapon System:** Automatic targeting of nearby enemies with multiple projectiles and damage upgrades.
- **Leveling & Upgrades:** Collect experience dots dropped by enemies to level up and choose between power-ups.
- **Infinite World:** A massive 4000x4000 world with strict boundaries and relative movement.
- **Highscore System:** Local storage-based highscore tracking with a 3-letter name entry system.
- **Optimized Performance:** Uses spatial partitioning and sprite pre-rendering for smooth 60 FPS gameplay.

## 🕹️ Controls

- **Movement:** WASD or Arrow Keys
- **Pause:** SPACE
- **Restart:** Click on the Game Over screen or press SPACE

## 🛠️ Tech Stack

- **Frontend:** HTML5 Canvas, Vanilla JavaScript (ES6+), Vanilla CSS
- **Development Tooling:** [serve](https://www.npmjs.com/package/serve) for local development

## 🚀 Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/philipboldt/neon-survivor.git
   cd neon-survivor
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run serve
   ```

4. **Play the game:**
   Open your browser and navigate to `http://localhost:3000`.
