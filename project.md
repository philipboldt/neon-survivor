# Neon Survivor

A functional game prototype where the player survives against waves of enemies in a neon-themed environment.

## Current Status
- [x] Initial Project Setup
- [x] Prototype Implementation (Single HTML File)
  - [x] Fullscreen Canvas
  - [x] Static Player (Center)
  - [x] Enemy Spawning (Outside screen)
  - [x] Vector Movement & Distance-based Acceleration
  - [x] Neon Visual Style (Glow effects)

## Tech Stack
- HTML5 Canvas
- Modern JavaScript (ES6+)
- Vanilla CSS

## Features
- **Player:** Blue glowing neon circle at the center of the screen.
- **Enemies:** Red glowing neon squares that move towards the center.
- **Mechanics:** Enemies accelerate as they approach the player, up to a maximum speed.
